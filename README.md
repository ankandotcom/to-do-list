# ⚡ Modern Minimalist To-Do

A high-performance, aesthetically pleasing To-Do application. Built with a focus on clean UI, fluid user experience, and modern CSS techniques.

**[Live Demo Link →](https://to-do-list-jade-gamma.vercel.app/)**

---

## 🚀 The Highlights
*   **Sleek Aesthetic:** Features a "Blueprint" grid system with a soft-blur fuchsia glow.
*   **Keyboard Optimized:** Swift task entry with `Enter` key support.
*   **Responsive:** Pixel-perfect layout from mobile to 4K displays.
*   **Zero Bloat:** Built with Vanilla JS for lightning-fast interactions.

## 🛠️ Tech Stack
| Tool | Usage |
| :--- | :--- |
| **Tailwind CSS** | Utility-first styling & layouts |
| **Vanilla JS** | Reactive DOM manipulation |
| **FontAwesome** | Intuitive UI iconography |
| **Vercel** | Continuous Deployment |
